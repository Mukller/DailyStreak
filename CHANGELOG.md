\## \[1.0.0] - 2026-05-14



\### Added

\- Initial public release of DailyStreak

\- Core streak tracking functionality

\- User authentication system

\- Dashboard with streak statistics

\- Notification system for daily reminders

\- Mobile-responsive design

\- Comprehensive documentation (README, CONTRIBUTING, LICENSE)

\- Code of Conduct for community contributors



\### Features

\- Track daily habits and streaks

\- Visualize progress with charts

\- Set custom streak goals

\- Community contributions support

\- Multi-language support (English, Russian)
